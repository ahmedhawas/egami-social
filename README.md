## Running and Testing

In order to run the app:
1. clone the repo.
2. Run `npm install`.
3. Run `npm start`.
4. Navigate to `localhost:3000` in your browser.

In order to run tests:
1. After cloning and install node modules.
2. Run `npm test`.
